# BouHockey(ボウホッケー)
  
## 概要説明
 専門学校入学後２ヶ月半の時期にグループ制作したホッケーゲームです。<br>
 <br>
 プレイヤー２人で対戦するホッケーゲームです。<br>
 ゲームの特徴としてステージ端の柵にパックがぶつかってバウンドすると、その数に応じてパックが進化して大きくなります。<br>
 銅→銀→金の３段階に進化して、グレードが高いほどゴールに入れたときの得点も上がります。<br>
 また、金のパックになると柵を破壊することができ、場外に飛び出しても得点になります。<br>
